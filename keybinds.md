|--------------|---------------------------------------------------|
| Ctrl + r     |                            Command history search |
| Ctrl + t     | Select file and directories, use TAB for multiple |
| Ctrl + c     |                        cd into selected directory |

## tmux

| Keybinding    |                            Description |
|---------------|----------------------------------------|
| Ctrl + Space  |                                 Prefix |
| Prefix + H    |            Resize pane + increase left |
| Prefix + J    |          Resize pane + increase bottom |
| Prefix + K    |             Resize pane + increase top |
| Prefix + L    |           Resize pane + increase right |
| Prefix + g    |                          Lazygit popup |
| Prefix + s    |                          Pane switcher |
| Prefix + c    |                              Copy Mode |
| Copy Mode + v |                        Begin selection |
| Copy mode + y |                         Yank selection |
| Prefix + p    |                                  Paste |

### tmux-tilish plugin

| Keybinding        |                            Description |
|-------------------|----------------------------------------|
| Alt + 0-9         |         Switch to workspace number 0-9 |
| Alt + Shift + 0-9 |             Move pane to workspace 0-9 |
| Alt + hjkl        |          Move focus left/down/up/right |
| Alt + o           |                  Move focus cyclically |
| Alt + Shift + o   |                   Move pane cyclically |
| Alt + Enter       |                        Create new pane |
